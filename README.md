# 剪贴板博物馆 🏛️

自动记录剪贴板历史的桌面应用。复制任何文字、链接、图片——自动存档，随时搜索回顾。

## 功能
- 📋 自动记录剪贴板文字/链接/图片
- ⭐ 收藏重要剪辑
- 📝 备忘录
- ⌨️ Alt+Shift+V 全局快捷键快速查看
- 🧹 自动清理（可设保留天数）
- 🔌 开机自启 + 系统托盘

## 技术栈
Electron + React + TypeScript + Vite + IndexedDB

## 开发
```bash
npm install
npm run dev:vite  # 启动前端开发
npm run dev:electron  # 启动 Electron
```
